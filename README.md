# nya短链  

##### 用于缩短链接  

## 依赖环境  

- PHP 5.4+  
- MySQL 5+  
- Apache 2+
- .htaccess和Mod_Rewrite支持  

## 安装  

1. 上传文件至网站目录  
2. 导入sql/sample.sql  
3. 修改config.sample.php为config.php，修改数据库连接信息
